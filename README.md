# cra-mobx

## 说明

* 用 [create-react-app](https://github.com/facebook/create-react-app) 创建
* 用 [react-app-rewired](https://github.com/timarney/react-app-rewired) 修改配置
* 配置了 mobx, less

## 使用

### 下载依赖

`npm install`

### 运行

`npm start`

### 构建

`npm run build`

### 跑生产

`npm run prod`

#### 详细文档 请看[CRA-README.md](./CRA-README.md)
